# subspace-express
subspace express js assignment

## Tasks

- **Data Retrieval**
- **Data Analysis**
- **Response**
- **Blog Search Endpoint**
- **Error Handling**
- **Caching**

## Endpoints

### Blog Statistics
```bash
curl --request GET \
  --url http://localhost:3000/api/blog-stat
```
### Blog Search
```bash
curl --request GET \
  --url http://localhost:3000/api/blog-search?query=flutter
```
